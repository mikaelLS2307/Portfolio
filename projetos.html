    Site Gela-Goela - Açaiteria

    Site desenvolvido durante a faculdade no 4º semestre na matéria de projeto integrador de API.

    Este é um projeto acadêmico desenvolvido por mim e mais duas colegas, onde o intuito seria realizar vendas e entrega de açaí, onde o usuário pode escolher os complementos, informar o CEP para a entrega, realizar cadastro e login, tudo isso através de API, que é um serviço que busca e envia os dados para o Banco. Neste caso, construimos uma API para cadastrar o usuário, para realização de login, e na tela de ADM tem disponível o CRUD, que seriam as 4 operações básicas de uma aplicação, sendo elas: Create, Read, Update e Delete. Ou seja, somente o ADM possui acesso a tela que contém essas operações que manipulam o banco de dados, como alterações nos dados ou exclusão.
